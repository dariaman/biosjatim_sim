Summary Produksi Tahunan

  1. Pada daftar menu, pilih menu Summary Produksi Tahunan

  2. Masukan data yang ingin ditampilkan, lalu tekan Search

  3. Data ditampilkan berupa diagram Pie. Tekan tombol Lihat Detail untuk menampilkan dalam bentuk tabel

  4. Sesuaikan filter untuk menampilkan data, lalu tekan Search

  5. Pada tampilan, geser layar untuk melihat keseluruhan data

  6. Untuk menampilkan data per-regional tekan Pilih Regional

  7. Data akan tampil berdasarkan regional yang dipilih
